FOC启动

2026年04月20日 电控 约9.9k字 预计需要57分钟

一、开环控制

Q&A：

1. PWM 频率与精度的关系

结论：设置目标电压时只看精度（分辨率），不直接看频率。

  • 精度 8 位（28=256）：意味着你把 PWM 占空比分成了 256 份。无论你的频率是 3000Hz 还是 30000Hz，代码里写的 255 永远代表 100% 占空比（全电压输出）。
  • 3000Hz 的意义：这是开关切换的速度。频率越高，电流纹波越小，噪音越小（3000Hz 其实在人耳听觉范围内，会有明显的啸叫声，工业通常用 10k-20kHz）。
  • 关系：频率决定了你能控制的“响应速度”，精度决定了你控制的“细腻程度”。

2. 扭矩和转速有关吗？

在开环状态下：有关，且是“此消彼长”的痛点。

  • 物理层面：P=T×ω（功率 = 扭矩 × 角速度）。电机输入的功率有限，转速越高，能提供的扭矩通常越小。
  • 开环痛点：在开环 FOC 中，你给固定的 Uq​（电压幅值）。当转速升高，电机产生的反电动势（Back EMF）也会升高。反电动势会抵消掉你输入的电压，导致实际电流减小。
  • 结果：转速越快，实际电流越小，扭矩就越弱。所以开环控制在高速时往往“没力气”。

3. Uq​ 控扭矩，电角度控转速？

是的，但在开环里这是“强行带跑”。

  • Uq​（电压幅值）：它决定了定子磁场的大小。磁场越大，吸力/推力越大，即扭矩越大。
  • 电角度（Phase）：在开环里，是你强行让定子磁场以某个速度旋转。转子因为磁力吸引，被迫跟着这个磁场跑。
  • 形象比喻：Uq​ 是你手里拉驴的绳子的粗细（力气），电角度的变换频率就是你跑的速度。你跑多快，驴就得跟多快。如果 Uq​ 太小而你跑太快，绳子断了（磁场步进），驴就丢了（电机失步抽搐）。

4. 为什么电角度要控制在 2π 内？

这是为了数学计算的连续性和防止溢出。

  • 周期性：电机的磁场分布是周期的。从 0 到 2π（360°）代表磁场转完了一个完整的电周期。2.1π 和 0.1π 在 sin 函数计算结果上是一模一样的。
  • 数值安全：如果你不重置，角度会无限增加。在单片机里，float 或 int 累加到最后会溢出，导致计算崩溃。所以每次超过 2π 就减去 2π。

5. 函数逻辑链条

你的理解基本正确，完整的开环逻辑链条如下：

  1. 输入目标角速度：ω (rad/s)。
  2. 计算机械增量：ΔAnglemech​=ω×Δt（Δt 为代码两次循环的时间间隔）。
  3. 累加电角度：
    Angleelec​=Angleelec​+(ΔAnglemech​×PolePairs)
  4. 限制范围：对 Angleelec​ 进行 2π 取模。
  5. 坐标生成：
    • 你有手动设置的 Uq​（想让电机多有力）。
    • Ud​ 通常设为 0。
  6. 坐标变换（或直接极坐标生成）：
    • 如果是 C-P 变换派：把 Uq​,Ud​,Angleelec​ 丢进反 Park 变换 → 反 Clarke 变换 → Ua​,Ub​,Uc​。
    • 如果是极坐标派：直接 Ua​=Uq​sin(Angleelec​) 等。
  7. 输出：将 Ua​,Ub​,Uc​ 映射到 PWM 占空比上。

你可能注意到了代码里克拉克逆变换有个奇怪的操作：

为什么要加 power_supply / 2？
这就是为了维持“恒等”。

  • 单电源局限： 你的电池是 12V 到 0V，没有负电压。但正弦波（ Uα,Uβ ）是有正有负的（比如 -4V 到 +4V）。
  • 抬升电平： 加上 VCC/2（6V）后，原本 -4V ~ +4V 的波形变成了 2V ~ 10V。这样三相占空比就永远落在 0~1 之间，电机才能工作。
  • 抵消效应： 虽然每相都加了 6V，但电机线圈受到的电压是相与相之间的压差（例如 Ua−Ub ）。两边都加了 6V，减法一做，这 6V 就抵消了。这在电学上叫**“抬升中性点电压”**。

8. Uq 的极限：为什么不能超过 supply / 2？

代码注释里提到：最大只能设置为 Uq = voltage_power_supply / 2。
这是由数学定义的。在克拉克变换中：

  • Ua=Uqsin⁡(θ)+Offset
  • 如果你想让 Ua 的最大值不超过 Vcc ，最小值不低于 0 。
  • 那么 Uq 的振幅（从中心往上摆动的幅度）最大只能是 Vcc 的一半。
    如果你设置 Uq=10V ，供电只有 12V ，那么加上 6V 的偏移后， Ua 理论上要输出 16V ，但 PWM 最高只能给到 12V ，波形就会变成**“削峰”的方波**，电机运行会变得非常粗暴、发热、噪音剧增。

一个需要注意的细节

在开环控制里，你是假设转子已经跟上了你的电角度。所以你的 Angle_elec 是“自嗨”的累加值。 当你进入闭环控制（真正的 FOC）时，Angle_elec 就不能自己乱加了，必须通过编码器实测转子的角度，然后在它的基础上加 90°（π/2）去生成磁场，这才是真正的“定向”控制。
现在的开环练习，本质上是在练习如何生成一个旋转磁场。
